# 🚗 Bluetooth Controlled RC Car

This project is a Bluetooth controlled RC car built using Arduino Uno and HC-05 Bluetooth module. The car is controlled wirelessly using a mobile application.

---

## 🔧 Features
- Forward, Backward, Left, Right movement
- Wireless control using Bluetooth
- Real-time response
- Simple and low-cost design

---

## 🛠️ Components Used
- Arduino Uno
- HC-05 Bluetooth Module
- Motor Driver (L298N / L293D)
- DC Motors
- Battery

---

## ⚙️ Working
The mobile application sends commands via Bluetooth to the HC-05 module.  
The module communicates with Arduino using serial communication.  
Based on received commands, Arduino controls the motor driver to move the car in different directions.

---

## 🎮 Controls
- F → Forward  
- B → Backward  
- L → Left  
- R → Right  
- S → Stop
